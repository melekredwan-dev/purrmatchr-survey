# Purrmatchr - Cat Adoption Survey

A multi-step cat adoption survey form. A modified take on freeCodeCamp's "Build a Survey Form" certification project.

## About

This project is a modified version of freeCodeCamp's "Build a Survey Form" certification project. Rather than a simple single-page form as expected by the original project requirements, this implementation features:

- Multi-step form with a prgoress indicator
- Pastel colour scheme with a light/dark mode toggle
- Real-time validation with custom error messages
- Local storage to preserve progress
- Full accessibility support (ARIA, keyboard navigation)
- Responsive, mobile-first design

## Tech Stack

- HTML
- CSS
- Vanilla JS

## Local Development

1. Clone the repository:
```bash
   git clone git@github.com:melekredwan-dev/purrmatchr-survey.git
```

2. Open `index.html` in your browser, or use a local server:
```bash
   # if you have python installed:
   python -m http.server 8000   # or: python3 -m http.server 8000
   
   # then visit http://localhost:8000
```
